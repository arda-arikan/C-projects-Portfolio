#My C Learning Journey 

Welcome to my portfolio. I am an Electrical & Electronics Engineering student at Erciyes University. Here, I share the projects I developed while learning C programming and algorithms.

Projects

1. Hangman Game (Adam Asmaca) 
A console-based word guessing game developed to understand string manipulation and search algorithms.
* *Key Concepts:* struct, string.h, Linear Search, Game Loop.
* *Features:*
  * Health/Life system (5 attempts).
  * Dynamic word masking.
  * Win/Loss detection logic.

2. Expense Tracker (Bütçe Takip Sistemi) 
A financial management tool to track personal expenses using file handling.
Key Concepts: File I/O (fopen, fprintf), struct, pointers.
Features:
  * Add new expenses (Category, Price, Date).
  * Save data permanently to a .txt file.
  * Calculate total expenses automatically.
